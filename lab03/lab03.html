Bootstrap คืออะไร?



- เป็นสิ่งที่นิยมมากในการพัฒนาเว็ปไซท์ โดยเป็น Framework หรือ รูปแบบคำสั่งสำเร็จรูป ของภาษา html,css,javascript
โดยใช้สำหรับพัฒนาเว็ปแบบ responsive และ mobile-first web sites. - เป็น free front-end framework ที่ง่ายและเร็วในการพัฒนาเว็ป



ประหวัติ Bootstrap



- Bootstrap ได้รับการพัฒนาโดย Mark Otto และ Jacob Thornton โดยเริ่มต้นที่ Twitter และเปิดเป็น open source สิงหาคม 2011 บน GitHub



ทำไมเราต้องใช้ Bootstrap



  • ใช้งานง่ายแค่มีพื้นฐานของ htmlและcss
  • สามารถตอบสนองบนมือถือได้เป็ยอย่างดี
  • สามารถใช้กับเบาร์เซอร์ได้ทุกแบบ
