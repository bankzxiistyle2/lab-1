การตั้งค่าเริ่มต้นของ Bootstrap



- ค่าเริ่มต้นของ Bootstrap คำสั่ง ใน form นั้น < input>, < textarea>, < select> จะใช้ class .form-control จะทำให้ width=100%



โดย form จะมี layout ต่างให้เลือกใช้

  • Vertical form (this is default)
  • Horizontal form (class .form-horizontal .control-label)
  • Inline form (class .form-inline .sr-only)

Inline form

การกำหนดรูปแบบ inline form เป็นการทำให้คอนโทรลต่างๆ ของฟอร์มอยู่ในแถวเดียวกัน โดยโดยปกติแล้วหากเราไม่กำหนดค่า .form-inline ในแท็ก <form> คอนโทรลต่างๆ ของฟอ์มจะอยู่คนละแถว หรือคนละบรรทัด เหมือนกับตัวอย่างแรก แต่หากเรากำหนดค่า .form-inliine เข้าไป คอนโทรลต่างๆ จะอยู่ในแถวเดียวกัน ในการทำ inline form นั้นเราจะต้องทำการกำหนดค่าความกว้างของคอนโทรล โดยการใช้สไตล์ซีทกำหนดค่า ซึ่งตัวอย่างจะใช้ width ในการกำหนดค่าความกว้างของคอนโทรล หากเราไม่กำหนดเบราเซอร์จะแสดงความกว้างเต็ม 100% ซึ่งจะไม่เห็นความแตกต่างในการใช้ .form-inline และหากเราไม่ต้องการให้แสดง <label> เราสามารถกำหนดคลาส .sr-only ให้กับ <label> ได้ ตัวอย่างการใช้งาน



Bootstrap Horizontal Form
